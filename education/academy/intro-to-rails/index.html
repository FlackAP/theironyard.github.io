A Basic Intro to Rails

What you will learn and do

This crash course will explore the capabilities of the Rails/PostgreSQL stack. You’ll learn the what and the why, and how to set up your machine to start development.

Curriculum Overview

  • Demo development workflow with Foreman, Git, Heroku, Migrations, environment files
  • Demo the stubbing and libraries like VCR that make testing so much simpler
  • Cool things you can do thanks to monkey patching
  • Present an overview of standout features in PostgreSQL
  • Weaknesses of Ruby that are fixed by using jRuby
  • Explain scalability in terms of web application parts, level of complexity in the stack, application concerns, number of moving parts and how you can simplify all of it dramatically with jruby and postgres

Excited? You’ve come to the right place.

Is this for you?

If you have experience in another programming language and want a seasoned pro to show you how to start digging into Rails, this is your ticket.

Are there prerequisite skills?

You should have experience developing web applications with another language and want to understand Rails and PostgreSQL as well or better than your current setup.
